# Data Folder Structure

This folder contains raw and processed datasets for the Airbnb analysis.

## Raw Data (not tracked in Git)

Download from Inside Airbnb or Kaggle and place files here:

```
data/raw/
├── Austin/
│   ├── listings_austin.csv
│   └── calendar_austin.csv
├── Dallas/
│   ├── listings_dallas.csv
│   └── calendar_dallas.csv
├── Nashville/
│   ├── listings_nashville.csv
│   └── calendar_nashville.csv
├── New York City/
│   ├── listings_newyork.csv
│   └── calendar_newyork.csv
└── Seattle/
    ├── listings_seattle.csv
    └── calendar_seattle.csv
```

**Note:** Raw CSVs are excluded from Git via `.gitignore` because they exceed GitHub's 100MB file size limit.

## Processed Data

Generated by running the pipeline scripts (01–05):

- `listings_cleaned.csv` — cleaned and merged listings from all 5 cities
- `calendar_monthly_agg.csv` — monthly aggregated calendar data for seasonality analysis

These are also excluded from Git due to size.
